# CampusCrate 📦

CampusCrate is a student-focused web platform that enables students within the same city or nearby institutes to rent, sell, or donate essential items such as books, electronics, lab tools, and hostel utilities.

## 🌱 Features
- Item listing (Rent / Sell / Donate)
- City-based discovery
- Filters by category and price type
- Item requests board
- Simple contact buttons
- Duplicate/suspicious listing detection (demo logic)
- Social impact simulation (reuse, money saved, waste reduced)

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript (No backend)

## 🚀 Deployment
This project is designed for deployment on **Netlify**.

## 🎓 Purpose
Demonstrates usability, sustainability, and social impact for student communities.
